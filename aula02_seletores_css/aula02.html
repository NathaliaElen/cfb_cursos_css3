Seletores CSS3

Seletores são usados para escolher quais elementos do HTML serão estilizados.

Exemplos mais usados

Universal (`*`) seleciona todos os elementos.

Elementos (`p`, `h1`)

Classes posso indicar vários elementos para usar essa classe (`.minhaClasse`)

IDs específico para determinado elemento(`#meuID`)

Mas existe vários outros.
