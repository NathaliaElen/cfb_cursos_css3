Margem e Centralização

A propriedade margin no CSS controla o espaço externo de um elemento, definindo a distância dele para outros elementos ao redor.


Margin é uma meta propriedade, como configurar?

Pode-se especificar margens diferentes para cada lado (topo, direita, baixo, esquerda) ou aplicar um valor único para todos.

Aplicar um valor único para todos.

2 valores: O primeiro valor define a margem superior e inferior. O segundo valor define a margem esquerda e direita.

3 valores: O primeiro valor define a margem superior. O segundo valor define a margem esquerda e direita. O terceiro valor define a margem inferior.

Como é meta propriedade, posso usar os valores de ada lado separadamente:

Margin-top

Margin-right

Margin-bottom

Margin-left
