Cores CSS3

No CSS3, as cores são usadas para definir o estilo visual dos elementos em uma página. Para isso, você pode usar diferentes formas de representar cores:

Nome das cores: Especifica cores básicas pelo nome, como red, blue, green, etc. Por exemplo, color: blue;

Cores Hexadecimais: Usam um código começando com # seguido de 6 caracteres (3 para vermelho, verde e azul). Por exemplo, color: #006A67;

Hexadecimal contraído: se tenho dois dígitos iguais para o R G B, posso usar somente um dígito de cada. Por exemplo, color: #F00;

RGB: Define a quantidade de vermelho, verde e azul em valores de 0 a 255. Por exemplo, color: rgb(255, 87, 51);

RGBA: Similar ao RGB, mas adiciona um valor de opacidade (transparência), onde 0 é totalmente transparente e 1 é opaco. Por exemplo, color: rgba(255, 87, 51, 0.5);

HSL: Representa as cores com três valores: matiz (hue), saturação e luminosidade. Por exemplo, color: hsl(327, 80%, 50%);

HSLA: Assim como o HSL, mas inclui um valor de opacidade. Por exemplo, color: hsla(207, 92%, 47%, 0.5);
